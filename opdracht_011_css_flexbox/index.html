Wat informatie over de opbouw van de eerste site die ik ooit heb gemaakt!


De code

Dit is voor het eerst dat ik HTML en CSS een website bouw. Het is onwijs leuk wanneer iets eerst net niet doet wat je voor ogen hebt en na lang puzzelen het uiteindelijk wel lukt! Soms is het gewoon een punt of een comma die net verkeerd staat in de code.

De opmaak

De opmaak van de de site vind ik zowel technisch als creatief. Technisch doormiddel van code eigenschappen toekennen aan onderdelen. Creatief door te kijken naar het geheel en of het er ook mooi uitziet.

De indeling

Wat komt waar en hoe ziet het er mooi, overzichtelijk maar ook niet standaard uit. leuk om mee te spelen. met kleuren kan je heel veel doen en vind ik ook belangrijk bij de indeling van de site.

Het totaal

Alles samen zorgt voor het totaalplaatje en voor de Gebruikers ervaring, user experience. Deze is erg belangrijk en als deze goed is dan kan je op intuitie de pagina makkelijk navigeren
